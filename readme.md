# Coursemate Forum

### 1. Create virtual environment (Optional)
    Windows: 
    py -3 -m venv venv
    venv\Scripts\activate

    Mac: 
    python3 -m venv venv
    venv/bin/activate

### 2. Install dependencies
    pip install -r requirements.txt

### 3. Run the server
    Windows: python app.py
    Mac: python3 app.py
